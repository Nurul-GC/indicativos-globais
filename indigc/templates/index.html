Introdução

Com exatamente 240 valores a IndiG API está composta por apenas 2 endpoints, correspondentes ao:

Retorno Total Dos Valores:
indigc.herokuapp.com/indicativos/

E ao Retorno De Um Valor Específicado Pelo Seu Nome:
indigc.herokuapp.com/pais/[nome_do_pais]/
Ou ainda, acesse a lista de paises disponibilizada aqui e requisite os valores individualmente de forma simples.

Formatos Válidos

A IndiG retorna dois formatos padrões:

JSON
Para retorno de um JSON, existem URLs expecificos que retornam os valores requisitados, como mostra a lista aqui em baixo.

API Responsiva
Para ter acesso aos valores de forma navegável e responsiva, basta abrir o URL da API num navegador.
